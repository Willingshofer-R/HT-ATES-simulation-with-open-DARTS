# HT-ATES-simulation-with-open-DARTS

A repository for simulating HT-ATES in the subsurface with the open-source Delft Advanced Terra Research Simulator (DARTS) v 1.3.2 with python 3.12.

=== File history and credits: 

The run_darts_model and basic_ates_model files are originally created by Yuan Chen (TU-Delft) 2025.

The run_darts_model and basic_ates_model are being modified by Robin Willingshofer(TU-Delft) 2026. The current version v2. implement as flexible number of wells as well as a flexible number of input soil layers. Efficient gridding is automated, based on well-coordinates and user-specified settings.
